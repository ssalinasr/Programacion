FrameWork: Marco de trabajo que consiste en un conjunto de elementos y estandares, enfocados en un problema especifico, que funciona como punto de referencia para desarrollar dicho problema

Dicho marco, representa una arquitectura de software que genera un modelo entre todos los elementos, creando una estructura y una metodologia para trabajar, que se encarga de exntender utilizar dichos elementos
